二、单一表格中检索数据

MySQL语句在执行时关键字是忽视大小写的，建议关键字使用大写。

同时也会忽略回车 空格，所以下方两个指令是一样的，但建议使用第二种。


1. SELECT 语句

  • USE 用来指定数据库
  • SELECT 用来查询具体的信息
  • FROM 指定了从某个表中查询
  • ORDER BY 可以用来排序

  • 读取指定列，如果需要查询列过多，可以换行。也可以直接在SELECT 语句中进行数学运算。
  • AS 可以指定名称。如果指定的名称包含空格，可以用引号包含起来。

  • 如果我们想检索出不重复的结果可以使用 DISTINCT 关键字
