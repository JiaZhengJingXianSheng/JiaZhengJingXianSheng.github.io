逻辑判断笔记整理

发表于2025-06-23|更新于2025-06-23|GWY笔记


前推后和后推前，其实本质上是重要的词在后面。

不重要的 -> 重要的。

前推后

  • 如果 ... ，那么 ...

  • 若 ... ，则 ...

  • 只要 ... ，就 ...

  • 所有 ... ，都 ...

  • 为了 ... ，一定 ...

  • ... 是 ... 的充分条件

速记：如果只要就，若凡则所有都，一定必须是充分。

A -> B = -B -> -A


后推前

  • 只有 ... 才 ...
  • 不 ... 不 ...
  • 除非 ... 否则不 ...
  • ... 是 ... 的基础/假设/前提/关键（谁必不可少谁在后）
  • ... 是 ... 的必要条件/必不可少的条件


"且"与"或"

A且B ： 和、并且、又、而且、虽然 ... 但是 ...

A或B ： 或、A和B至少有一个
